## 3. How to View
- Open the relevant folder above
- Play any `.webm` file in a modern browser or video player

---

## 4. Generation Details
- Videos are generated automatically by Playwright using the `video` option in the config
- Each sequence is ~1 minute, simulating a real user journey
- Demo and live configs: `scratch/playwright.demo.config.js`, `scratch/playwright.live.config.js`
